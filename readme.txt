This plugin adds a widget to your dashboard showing every post which has a comment you did not respond to

== Description ==

With this plugin you get a new widget on your dashboard showing you every post on which 
you haven't commented as last person. This is very usefull for blogger who wants to give
the attention to their commenting readers they should have.

So if you try to reply on every comment because its good behaviour in your sight this plugin
will help you not to miss one of them.

== Installation ==

1. Upload the plugin directory from the zip-file to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. See your new widget on the dashboard
